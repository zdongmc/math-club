# MCPS Mathematics League - Meet Timer Slides

Interactive presentation slides with timers and voice announcements for running Math League meets.

## File

`math-league-meet-slides.html` - Complete standalone HTML file with embedded timers and voice announcements

## Overview

These slides guide students through all three rounds of a Math League meet with automatic timing and audio announcements.

## Meet Structure

**Total Time**: ~60 minutes
**Total Possible Points**: 64

### Round Breakdown

1. **Team Round** - 20 minutes, 6 questions, calculators allowed (12 points max)
2. **Relay Team Round** - 16 minutes total, two 8-minute relays, no calculators (16 points max)
3. **Individual Round** - 24 minutes, 6 problems, no communication (36 points max)

## Slide Contents

1. **Welcome Slide** - Meet dates, backpack instructions, round overview
2. **Team Round Instructions** - Rules and format
3. **Team Round Timer** - 20-minute countdown
4. **Relay Round Instructions** - Rules and format (includes calculator reminder)
5. **Relay Round 1 Timer** - 8-minute countdown with bonus tracking
6. **Relay Round 2 Timer** - 8-minute countdown with bonus tracking
7. **Individual Round Instructions** - Rules and ARML tracking info
8. **Individual Round Timer** - 24-minute countdown
9. **Meet Complete** - Summary and parent portal reminder

## Timer Features

### Voice Announcements

All timers include automatic voice announcements at key intervals:

- **Start**: "[Round name] begins. You have [X] minutes."
- **Halfway point**: Announces remaining time
- **5 minutes**: Warning announcement
- **2 minutes**: Warning announcement
- **1 minute**: Final minute warning
- **10 seconds**: "10 seconds remaining" + countdown (10, 9, 8, 7...)
- **Time's up**: "Time is up. Please stop working and submit your answers."

### Relay Round Special Announcements

- **4 minutes**: "4 minutes remaining. Bonus window has ended." (2 points only from this point)

### Visual Indicators

- **Normal**: White display on blue background
- **Warning** (halfway): Yellow/orange pulsing animation
- **Critical** (final 10 seconds): Red pulsing animation

## Controls

### Timer Controls (on each timer slide)
- **Start Timer** - Begins countdown and announces round start
- **Pause** - Pauses/resumes timer
- **Reset** - Resets timer to initial time

### Navigation
- **Navigation buttons**: Bottom right corner (◄ ►)
- **Keyboard shortcuts**:
  - `←` / `→` arrows: Navigate between slides
  - `Space bar`: Start timer on current timer slide

## Usage Instructions

1. Open `math-league-meet-slides.html` in a web browser
2. Use full-screen mode (F11 or browser full-screen) for best experience
3. Ensure speakers/audio is enabled for voice announcements
4. Navigate through slides using arrow buttons or keyboard
5. Start timers when students are ready for each round

## Important Notes

- **Calculator policy**: Only allowed during Team Round (first round)
- **Relay rounds**: Students must put away calculators before relay rounds begin
- **Individual round**: No communication allowed between students
- **Backpack storage**: Students should store backpacks at front of room before meet begins
- **Results**: Remind students that results will be posted on the Parent Portal

## Technical Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- JavaScript enabled
- Web Speech API support (for voice announcements)
- No internet connection required (fully self-contained HTML file)

## Troubleshooting

**Voice announcements not working:**
- Check browser permissions for speech synthesis
- Ensure volume is turned up
- Try Chrome or Edge (best speech synthesis support)

**Timer not starting:**
- Ensure JavaScript is enabled
- Refresh the page and try again

**Slides not displaying correctly:**
- Use a modern browser
- Try full-screen mode
- Refresh the page
